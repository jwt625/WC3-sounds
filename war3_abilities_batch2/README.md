## Difference from Other Sound Folders

### vs. war3_mpq_additional/Units/Combat
- **Combat/** = Generic impact sounds (metal on flesh, wood on stone, etc.)
- **Weapons/** = Specific weapon projectile sounds (arrow whoosh, rifle shot, etc.)

### vs. sounds/ (from sounds-resource.com)
- **sounds/** = Unit voices and basic UI sounds
- **Spells/** = Ability activation and spell effect sounds

## File Types

- Most files are in WAV format (uncompressed, high quality)
- Some files may be in MP3 format
- All visual effect files (.mdx, .blp) have been excluded

## Usage Notes

These sounds include:
- Spell casting sounds (Blizzard, FlameStrike, Heal, etc.)
- Hero ability sounds (Metamorphosis, Death Coil, etc.)
- Item usage sounds (some items have sound effects)
- Weapon projectile sounds (arrows, rifles, catapults, etc.)
