# War3.mpq Additional Sounds

Downloaded from: https://github.com/WarRaft/War3.mpq

Total files: **2,823 audio files** (WAV and MP3)

## Directory Structure

### 1. Units/ (175 files)
- **Combat/** (154 files) - Weapon impact sounds
  - Metal/Wood/Rock/Axe impacts on Flesh/Metal/Stone/Wood
  - Heavy/Medium/Light variations with Bash/Chop/Slice actions
  - Examples: `MetalHeavyBashFlesh1.wav`, `WoodMediumChopWood1.wav`
- **Footsteps/** (21 files) - Unit movement sounds

### 2. Interface/ (135 files)
- Basic UI sounds: Click, Error, Hint, MouseClick, etc.
- Quest sounds: QuestCompleted, QuestFailed, QuestNew
- Game events: GameFound, GamePause, ItemReceived, PickUpItem
- **Warning/** (77 files) - Faction-specific warning sounds
  - Human, Naga, NightElf, Orc, Undead warning subdirectories

### 3. Dialogue/ (2,438 files)
- **Campaign dialogue for all races:**
  - HumanCampaign
  - OrcCampaign
  - NightElfCampaign
  - UndeadCampaign
- **Expansion campaign dialogue:**
  - HumanExpCamp
  - OrcExpCamp
  - NightElfExpCamp
  - UndeadExpCamp
- **Additional:**
  - TutorialCampaign
  - GenericWarnings (8 MP3 files)
  - Extra dialogue

### 4. Ambient/ (73 files)
- **Zone-specific ambient loops:**
  - Ashenvale, Barrens, BlackCitadel, CityScape
  - DalaranRuins, IceCrown, LordaeronFall, LordaeronSummer
  - LordaeronWinter, Northrend, SunkenRuins
- **DoodadEffects/** (54 files) - Environmental effects
  - Gate sounds, water loops, earthquakes, portals
  - Campaign-specific sounds: Frostmourne chants, Eye of Sargeras
  - Examples: `GateOpen.wav`, `Earthquake.wav`, `WaterWaterFallLoop1.wav`
- **Faction theme loops:** Alliance, Human, Night Elf, Orc, Undead

### 5. Time/ (2 files)
- DaybreakRooster.wav - Morning transition sound
- DuskWolf.wav - Evening transition sound

## Key Sound Categories

### Combat & Weapon Sounds (154 files)
All the weapon impact variations you'd need for combat:
- Metal weapons hitting different surfaces
- Wood weapons (clubs, staves)
- Rock impacts
- Ethereal/magical impacts

### Warning & Notification Sounds (85 files)
- Generic warnings: Gold collapsed, gold mine low, hero fallen, no mana, town/unit under attack
- Faction-specific warnings for each race
- Research/upgrade completion sounds

### Environmental & Ambient (127 files)
- Zone ambiance for different terrain types
- Interactive environmental sounds (gates, water, earthquakes)
- Campaign-specific cinematic sounds

### Campaign Dialogue (2,438 files)
Complete campaign voice lines for all four races across base game and expansion.

## Usage Notes

- Most files are in WAV format (uncompressed)
- Some interface sounds and dialogue use MP3 format
- These sounds complement the unit voices in the main `sounds/` folder
- Campaign dialogue provides narrative context and character voice lines
